# 2Dゲームの基礎

## Tilesetの使い方

### 概要

TilesetはGodotの標準機能で、

ここでは、resourcesフォルダ下にtilesetsフォルダを作成し、その中にリソースファイルを入れることにする。

### 操作例

#### Tilesetの作成

ファイルシステムドック上のtilesetsフォルダ上で右クリック、「新規作成」→「リソース」→「Tileset」を選択。

作成する.tresファイルのインスペクターを確認、タイルごとのサイズを設定する。

「タイルソース」タブ上に画像をD&Dし、使用する画像を追加。

セットアップで、Texture Region Sizeにタイルごとのサイズを設定する。「マージン」「分離」は画像ごとの間隔で、通常は0で良い。
