# ピクセルパーフェクトなゲームを作るための設定

## プロジェクト設定

「プロジェクト」→「プロジェクト設定」→「サイズ」→「ビューポートの幅」・「ビューポートの高さ」でビューポートを変えられるが、右上の「高度な設定」をONにすると、それとは別にウィンドウサイズを変えられる。
ドット絵の場合、「ビューポート」は小さい解像度で作成しておき、それとは別に大きいウィンドウサイズを設定する。

「高度な設定」をONにした後、「プロジェクト」→「プロジェクト設定」→「ウィンドウ広さのオーバーライド」「ウィンドウ高さのオーバーライド」に値を設定する。また、「ストレッチ」の「スケールモード」をintegerにする。


## テクスチャの設定

画像をD&Dで画面上に配置した場合、

- CanvasItem → Texture → Filter を「Nearest」に設定する。
