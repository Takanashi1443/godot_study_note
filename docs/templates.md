# テンプレート

## Godotにおけるプロジェクトのテンプレート

GodotのプロジェクトはUnityと比べて以下のような性質があり、プロジェクトのテンプレートを持っておくのが容易である。

- フォルダを作成する際にメタファイルなどは作られない。
- 所定の内容の、project.godotファイルがあるフォルダがプロジェクトのルートと認識される。

本リポジトリにも、templatesフォルダ内にGodotプロジェクトのテンプレートが保存されており、コピペ・名前を変更することで簡単にプロジェクトをセットアップできる。

## project.godotについて

project.godotにはプロジェクト設定が入る。例えば、

- プロジェクト名
- project.godotファイルの形式のバージョン
- Main Scene
- Input Map

などが保存されている。
